# A/B Test Executive Decision Dashboard

## Overview
This project analyzes the business impact of a **10% discount experiment** using A/B testing methodology.  
The goal is to determine whether the discount increases **revenue per user (RPU)** and if the result is **statistically significant** enough to support a full rollout.

## Key Results
- **RPU uplift:** +1.9% vs. control  
- **p-value:** < 0.001 (statistically significant)  
- **95% confidence interval:** +0.77% to +2.99%  
- Entire confidence interval lies **above zero**, confirming a **positive and reliable revenue impact**
